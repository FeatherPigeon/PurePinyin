# PurePinyin 极简拼音转换器
一个基于 Python 开发的轻量级汉字转拼音工具。它专注于提供最纯净的转换体验：自动剥离所有标点符号，并将多行文本合并为连贯的拼音流。
## ✨ 功能特性
- 🚀 极简转换：自动过滤掉所有中英文标点符号、特殊字符，只保留纯粹的拼音。
- 📦 长文友好：支持多行输入，并自动将结果合并为一段，每字拼音间以空格分隔。
- 📋 一键复制：内置一键复制到剪贴板功能。
- 🧹 快捷清空：支持一键重置输入输出区。
## 📖 使用示例
- 输入：\
    你好，世界！这是一个测试。
- 输出：\
    ni hao shi jie zhe shi yi ge ce shi
